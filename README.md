# 📚 Library Management System (CLI)

A command-line Library Management System built with Python for managing books, borrowing, and returns through a simple terminal interface.

> **Project Status:** 🚧 In Development

---

## 📖 Overview

This project is a command-line application designed to manage a library's collection of books and basic borrowing operations. The focus is on building a clean, modular, and maintainable application with a user-friendly terminal interface.

---

## ✨ Planned Features

- Add new books
- View all books
- Search books
- Update book information
- Remove books
- Borrow books
- Return books
- Track book availability
- Input validation
- Menu-driven CLI

---
